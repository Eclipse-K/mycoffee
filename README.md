# 마이커피(My-Coffee)

> **Your Coffee, Your Choice**  
> 고유의 커피 경험을 제공하기 위해 만들어진 커스터마이징 가능한 커피 플랫폼.

---

## 📖 **프로젝트 소개**

**My-C**는 사용자에게 최적의 커피 쇼핑 경험을 제공하는 데 중점을 둔 웹 애플리케이션입니다.

- **구매 간소화**: 쉽고 빠른 구매 프로세스
- **사용자 맞춤형 옵션**: 사용자의 선호도에 따라 다양한 선택 제공
- **깔끔하고 직관적인 UI**: 누구나 사용할 수 있는 편리한 인터페이스

---

## 🚀 **주요 기능**

- **상품 카테고리 분류**: Whole Bean, Drip Bag 등 다양한 옵션 제공
- **장바구니 기능**: 상품 추가, 수정 및 삭제
- **구매 페이지**: 배송 요청사항 입력 및 선택, 실시간 가격 계산
- **사용자 계정 관리**: 로그인 후 마이페이지로 개인화된 정보를 확인 가능
