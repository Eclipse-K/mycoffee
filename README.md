# My Coffee

**Website:** [My Coffee](https://mycoffee-k.netlify.app)

## 소개

My Coffee는 커피 애호가들을 위한 웹사이트로, 다양한 커피의 정보와 추천을 제공합니다. 사용자는 자신의 커피 취향을 발견하고, 커피에 대한 깊이 있는 지식을 얻을 수 있습니다.

## 특징

- 다양한 커피 종류에 대한 상세한 설명.
- 개인 취향에 맞는 커피 추천 기능.
- 모바일 및 데스크톱에서 최적화된 반응형 디자인.

## 기술 스택

- **Frontend:** HTML, CSS, JavaScript
- **Hosting:** Netlify
